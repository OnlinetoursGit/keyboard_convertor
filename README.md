Если пользователь забыл переключить раскладку, то можно сконвертировать то что ввёл.

    require 'keyboard_convertor'

    '<ht;ytd Ktjybl Bkmbx'.convert_keyboard()

    или

    '<ht;ytd Ktjybl Bkmbx'.convert_keyboard({ from: 'ru', to: 'en'})
